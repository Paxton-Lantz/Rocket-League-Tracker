Setup — 2 Steps, Done Forever

1
Download and run rl-capture.exe

Download it from the GitHub Releases page. Double-click it. A small icon appears in your system tray (bottom-right of your taskbar). It starts automatically every time Windows boots — you never have to think about it again.

2
Type your Epic username above the log form

Go to the Tracker tab. Find the "Epic username" box above the log form and type your exact in-game name. It saves automatically. You only do this once.

Epic username YourEpicName
✓
That's it — you're done

Start a session, play Rocket League. After each game the form fills in automatically within a second of the scoreboard appearing.

Something not working?

Status shows "Capture Offline"
rl-capture.exe isn't running. Double-click it again — it may not have started automatically yet on a fresh install.
Running but stats don't fill in
Double-check the Epic username box — it must match your exact in-game name (e.g. xX_ProPlayer_Xx). Case doesn't matter. Club tags are optional.
Stats fill but all zeroes / wrong numbers
The scoreboard wasn't fully loaded when captured. This is rare — just correct the numbers manually before hitting Enter.
Rocket League is on a second monitor
Create a config.json file next to rl-capture.exe with the contents: {"monitor": 2}
