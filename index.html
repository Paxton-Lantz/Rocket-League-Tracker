One-Time Setup

Recommended

Using rl-capture.exe

No Python, no Tesseract — just two files.

  1. Download rl-capture.exe from the GitHub Releases page
  2. Place config.json in the same folder as the exe
  3. Open config.json and set "username" to your Epic display name
  4. Double-click rl-capture.exe — leave the window open while you play

From source

For building the exe yourself or running the Python script directly.

  1. Install Python 3 and Tesseract OCR (UB-Mannheim build)
  2. Run install.bat — installs Python deps and detects your monitors
  3. Edit config.json with your username
  4. Run start.bat before each session (or build.bat to create your own exe)

config.json

{
  "username": "YourEpicName",
  "monitor":  1
}
username Your Epic Games display name. Case-insensitive. Club tags optional.
monitor 1 = primary screen, 2 = secondary. install.bat sets this automatically if you have multiple monitors.
brightness_threshold Default 70. Raise it if captures are missed; lower it if the daemon fires during active gameplay.

Every Session

1
Run rl-capture.exe

Leave the terminal window open. Use the Test connection button above the log form to confirm it's running.

2
Start a session

Click "Start Session" and enter your current MMR. The status pill turns green once the daemon is detected.

3
Play — stats fill automatically

After each game, the form pre-fills within a second of the scoreboard appearing. Review and hit Enter to log it.

Troubleshooting

Pill shows "Capture Offline"
rl-capture.exe isn't running. Start it and wait a second, then hit Test connection.
Daemon running but stats never fill
The brightness pre-check is blocking OCR. Try raising brightness_threshold in config.json (default 70 — try 85 or 100).
OCR fires during active gameplay
Lower brightness_threshold (try 50 or 40). The end screen is darker than normal gameplay.
Stats fill but username not found
Check spelling in config.json. The daemon does a case-insensitive substring match — try just your base name without a club tag.
Capturing from the wrong screen
Set "monitor": 2 in config.json. Or run install.bat again — it will prompt you to choose.
